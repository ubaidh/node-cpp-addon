# node-cpp-addon


### Installation

 requires [Node.js](https://nodejs.org/) to run. gcc python 2.7

Install Python 2.7 (v3.x.x is not supported), and 
```sh
$ run npm config set python python2.7 
```
```sh
$ cd  
$ npm install 
$ npm run compile 
```
to build or 
``` sh
node-gyp configure build 
```
$ node app
```
$ npm start
```
